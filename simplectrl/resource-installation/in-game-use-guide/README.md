# 🎮 In Game Use Guide

### Controls
| Key              | Description          |
|------------------|----------------------|
| DOWN ARROW       | Hazard Key           |       
| LEFT ARROW       | Left Blinker Key     |
| RIGHT ARROW      | Right Blinker Key    |
| LEFT SHIFT KEY   | Blinker Modifier Key |
| T                | Engine Toggle Key    |
| RIGHT SHIFT KEY  | Shuffle Key          |
| E                | Refuel Key           |
| I                | Park Key             |
| LEFT CONTROL KEY | Park Modifier Key    |
