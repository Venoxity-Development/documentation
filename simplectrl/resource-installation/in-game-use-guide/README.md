# 🎮 In Game Use Guide

### Controls
| Key             | Description             | Type               |
|:---------------:|-------------------------|--------------------|
| DOWN ARROW      | Activates Hazard Lights | Quick Press        |   
|        E        | Initiates Refueling     | Quick Press        |
|        I        | Engages Parking Mode    | Quick Press        |
| LEFT ARROW      | Activates Left Blinker  | Quick Press        |  
| LEFT CONTROL KEY| Modifier for Parking    | Modifier Key       |
| LEFT SHIFT KEY  | Modifier for Blinker    | Modifier Key       |
| RIGHT ARROW     | Activates Right Blinker | Quick Press        | 
| RIGHT SHIFT KEY | Shuffle Seats           | Double Quick Press |
|        T        | Toggles Engine On/Off   | Long Press         |

### Commands
```Hood``` opens the front cover of the car.

```Trunk``` opens the rear storage compartment of the car.

```Shuffle``` same as ```RIGHT SHIFT Key```, enabling passengers to swiftly switch seats.
